Project Overview

Why Flow Automation?

Built to streamline and accelerate Google Flow tasks with intelligent browser control and real-time execution monitoring.

Why Built

Google Flow requires continuous manual inputs, repeated prompt copying, and constant browser tab switching. Flow Automation was built to eliminate tedious manual overhead and enable multi-task execution hands-free.

Problem Solved

Solves session interruptions, rate-limit stalls, missing queue states, and manual download tracking by orchestrating Playwright headless instances and providing instant error recovery.

Who It Is For

Designed for developers, AI prompt engineers, content creators, and power users who generate volume tasks on Google Flow and require clean, structured execution and analytics.

Designed to automate repetitive Google Flow tasks while providing a clean interface and analytics.

Setup Guide

How To Use

Follow these simple steps to deploy and run Flow Automation on your local machine.

1
Download Repository
Clone the repository or download the source ZIP file directly from GitHub.
git clone https://github.com/SaMGit06/FlowAutomation.git
2
Install Python
Ensure Python 3.9 or higher is installed on Windows with "Add Python to PATH" enabled.
3
Install Requirements
Install Flask, Playwright, and required Python helper packages.
pip install -r requirements.txt
4
Run server.bat
Double-click server.bat in the project root folder to start the local automation server.
5
Open Flow Automation
Navigate to http://localhost:5000 in your web browser.
6
Paste Prompts
Paste your batch prompts or import text files into the prompt queue box.
7
Click Run
Hit the Run button and sit back as Playwright executes your Google Flow workflow automatically!
